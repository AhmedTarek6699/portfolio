# Project Images

## How to Add Multiple Project Images

This project supports a carousel/slider with multiple images! You can add up to 10 images per project.

### Adding Images:

**Option 1: Single Image**
- Name your image: `image.jpg` or `image.png`

**Option 2: Multiple Images (Carousel)**
- Name your images: `image1.jpg`, `image2.jpg`, `image3.jpg`, etc.
- Or: `image1.png`, `image2.png`, `image3.png`, etc.
- The carousel will automatically detect and display all images
- Viewers can slide through images using arrow buttons

### Supported Formats:
- JPG/JPEG
- PNG
- GIF
- WebP

## Recommended Image Specifications

- **Dimensions**: 600x400 pixels (or similar 3:2 aspect ratio)
- **File size**: Under 500KB per image for faster loading
- **Total images**: Up to 10 images per project

## Example File Structure:
```
project/
  ├── image1.jpg  (First slide)
  ├── image2.jpg  (Second slide)
  ├── image3.jpg  (Third slide)
  └── README.md
```
