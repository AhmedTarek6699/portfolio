# Project 10 Images

## How to Add Your Project Image

1. Place your project image in this folder
2. Name it `image.jpg` or `image.png`
3. The image will automatically appear on your portfolio website

## Recommended Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: 600x400 pixels (or similar 3:2 aspect ratio)
- **File size**: Under 500KB for faster loading
